# Apache CXF (apache-cxf)
Apache CXF is an open-source services framework that helps build and develop web services using frontend programming APIs like JAX-WS and JAX-RS. It supports SOAP, REST, and CORBA services with full WS-* standards support.

## APIs

### Apache CXF
CXF provides Java APIs for building both SOAP (JAX-WS) and REST (JAX-RS) web services, with support for WSDL-first and code-first development, WS-Security, WS-ReliableMessaging, and automatic client/server code generation.
